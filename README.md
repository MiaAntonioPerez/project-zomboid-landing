# Project Zomboid Landing

A post-apocalyptic-styled fan landing page concept for Project Zomboid. Static HTML/CSS, no build step.

Implemented from a [Claude Design](https://claude.ai/design) project ([source](https://claude.ai/design/p/bb9abe63-f5ba-4bda-9649-904de06a4c88?file=Project+Zomboid+Landing.dc.html)).

Not affiliated with The Indie Stone.

## Run locally

Just open `index.html` in a browser, or serve it:

```sh
npx serve .
```
